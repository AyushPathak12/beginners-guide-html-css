HTML & CSS Quick Reference

HTML

Basic Elements

<html>...</html>
HTML document
<head>...</head>
Meta, or head, information (title tag, meta tags, etc.) not visible on the actual page
<body>...</body>
HTML body, the visible portion of the document

Text Elements

<h1>...</h1>
The largest heading
<h6>...</h6>
The smallest heading
<em>...</em>
Text with stressed emphasis
<strong>...</strong>
Text with strong importance
<p>...</p>
A paragraph

List Elements

    <ol>...</ol>
    An ordered list
    <ul>...</ul>
    An unordered list
    <li>...</li>
    A list item within an ordered or unordered list

    Other Common Elements

    <div>...</div>
    A generic block level container
    <span>...</span>
    A generic inline level container
    <br>
    A line break
    <img>
    An image
    <table>...</table>
    A table
    <tr>...</tr>
    A table row
    <td>...</tr>
    A table cell

    Common Attributes

    <img src="headshot.jpg">
    An image’s source
    <a href="http://wileecoyote.com">...</a>
    A link to a URL
    <div class="container">...</div>
    A class styling hook
    <link rel="stylesheet" href="http://wileecoyote.com/styles.css">
    Connecting an HTML document to a stylesheet
    <input type="text">
    A text input form field

    Basic HTML Document

    Also see, HTML & CSS Document Templates.

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a website.</p>
  </body>
</html>

    CSS

    p { ... }
    A type selector, in this case selecting a paragraph element
    .safe {...}
    A class selector, in this class selecting any element with the class attribute value of ‘safe’
    #dangerous {...}
    An ID selector, in this case selecting any element with the ID attribute value of ‘dangerous’
    div .safe {...}
    A compound selector, in this case selecting any element with the class attribute value of ‘safe’ with a division parent element
