Lesson 1

Terminology, Syntax, & Introduction

Homework

  1. 1 Read the first two lessons on A Beginner’s Guide to HTML & CSS
  2. 2 Create a new HTML document and markup a resume using basic elements, such as h1 and p element
  3. 3 Create a new CSS stylesheet, including a reset
  4. 4 Add a reference from your HTML document to your CSS stylesheet, connecting the two
  5. 5 Validate both your HTML document and CSS stylesheet using the W3C validation services
  6. 6 Watch the first week of videos on 30 Days to Learn HTML & CSS
  7. 7 Create an account on GitHub and attempt to install Git on your computer

Lesson 2

Elements & Semantics

Homework

  1. 1 Revisit the last lessons resume, refactoring it into more semantic markup using the new elements introduced within this lesson
  2. 2 Adding to the Resume, shaping it more into an About page, write a brief overview of yourself and rename the file to about.html
  3. 3 Create a Journal page named journal.html and add a journal entry detailing what you’ve learned thus far
  4. 4 Create a Home page named index.html and tease both your About and Journal pages
  5. 5 Create a global navigation on every page, linking them all together
  6. 6 Push your homework to GitHub for review
  7. 7 Read next weeks lesson on A Beginner’s Guide to HTML & CSS
  8. 8 Watch the second week of videos on 30 Days to Learn HTML & CSS
