Wile E. Coyote's Cheat Sheet

HTML

Basic Tags

  • <html>...</html> An HTML document.
  • <head>...</head> Header information (title, meta tags) that are not visible on the page.
  • <body>...</body> The visible portion of the document.

Text Tags

  • <h1>...</h1> The largest headline.
  • <h6>...</h6> The smallest headline.
  • <em>...</em> Emphasized text.
  • <strong>...</strong> Text with strong importance.

Organization Tags

  • <p>...</p> A paragraph.
  • <ol>...</ol> An ordered list.
  • <ul>...</ul> An unordered list.
  • <li>...</li> A list item within an ordered or unordered list.
  • <div>...</div> A division. Used as a generic block of HTML.

Other Common Tags

  • <img> An image.
  • <br> A line break.
  • <ht> A horizontal rule.
  • <span>...</span> A generic inline container.
  • <table>...</table> A table.
  • <tr>...</tr> A table row.
  • <td>...</tr> A table cell.

Common Attributes

  • <img src="http://wileecoyote.com/headshots.jpg"> An image.
  • <a href="http://wileecoyote.com>...</a> A link to a URL.
  • <div class="container">...</div> A division with a CSS class applied.
  • <link href="http://wileecoyote.com/styles.css" rel="stylesheet" type="text/css"> Connecting an HTML document to a stylesheet.
  • <input type="text"> A text form field.
  • <button>...</button> A button.

Setting Up A Basic HTML Document

<!DOCTYPE html> <html lang="en"> <head> <meta charset="utf-8"> <title>Hello World</title> </head> <body> <h1>Hello World</h1> <p>This is a website.</p> </body> </html>

CSS

  • Select a <p> element. p { ... }
  • Select the <p> elements on a page and change the text color and font size of that content. p { color: #ff0; font-size: 16px; }
  • Select any elements with the safe class applied, and make the background green. .safe { background-color: green; }
  • Select an element with the dangerous ID, and make the background red. #dangerous { background-color: red; }
  • Select div elements with the safe class applied, and make the background green. div .safe { background-color: green; }

Linking Files

  • Connecting an HTML document to a stylesheet hosted on another server. <link href="http://wileecoyote.com/styles.css" rel="stylesheet" type="text/css">
  • Connecting an HTML document to a stylesheet hosted on the same server, in the same directory as the HTML file. <link href="styles.css" rel="stylesheet" type="text/css">
  • Connecting an HTML document to a stylesheet hosted on the same server, but in a directory below the HTML file. <link href="css/styles.css" rel="stylesheet" type="text/css">
  • Displaying an image hosted on the same server, but in a directory below the HTML file. <img src="images/wileecoyote.jpg">
